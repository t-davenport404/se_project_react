# WTWR (What to Wear?): Front End

WTWR is a responsive, single-page web application that provides real-time local weather data and dynamic wardrobe recommendations based on the current temperature. This repository contains the client-side React code for the application.

## 🔗 Project Links

To explore the live application or view the backend infrastructure, check out the resources below:

- **Live Deployed Frontend:** [what2wear.tenspot.net](https://www.what2wear.tenspot.net)
- **Backend GitHub Repository:** [se_project_express](https://github.com/t-davenport404/se_project_express)
- **Live Deployed Backend API:** [api.what2wear.tenspot.net](https://api.what2wear.tenspot.net)

---

## 💻 Running the Project Locally

To test the frontend application locally:

1. Clone this repository to your local machine.
2. Run `npm install` in the project root folder to install all required dependencies.
3. Start the local development server using `npm run start`.
4. Open `http://localhost:3000` in your browser to view the app.

_Note: To test full functionality including user authentication and clothing uploads locally, ensure your backend server is running concurrently on port 3001._

## 🛠️ Technologies Used

- **Frontend Framework:** React.js, Functional Components, Custom Hooks
- **State Management & Routing:** React Context API, React Router DOM
- **Styling & UI:** HTML5, CSS3, BEM Methodology, Responsive Design (Mobile-First)
- **Data Fetching & APIs:** Native Fetch API, Weather API Integration
- **Build Tools & Package Management:** NPM, Webpack / Create React App
