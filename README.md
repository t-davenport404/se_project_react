# Project: SE Project React WTWR

## Description

WTWR is an app that provides up to date weather information and clothing recommendations. The clothing recommendations are based on the weather data received.

## Tech Stack

- HTML
- CSS
- Responsive Design
- JavaScript
- API
- NPM
- React
